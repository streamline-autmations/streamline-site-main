## Recommended design workflow

1. read `AGENTS.md`
2. read `CLAUDE.md`
3. read `streamline-site` skill
4. if needed, read `cuberto-design-review`
5. implement inside `src/site/*`
6. verify visually on desktop and mobile

## Red lines
- no stock or AI mockups on live site
- no fake testimonials
- no orange/dark-theme regressions
- no drift away from the single-purple system
