# Streamline Automations — Marketing Site

**streamline-automations.co.za** | Christiaan Steffen

## Stack
React 18 + TypeScript · Vite 5 · Tailwind CSS · Framer Motion · GSAP + Lenis · Supabase · Netlify

## Dev
```bash
npm install
npm run dev
```

## Direction
Cuberto-level motion craft on a white-minimal + single-purple-accent canvas.
Live site runs from `src/site/` (flag: `VITE_SITE_V2=true`).
Brand truth lives in `CLAUDE.md` and `.claude/skills/streamline-site/SKILL.md`.

## Deploy
Auto-deploys to Netlify from `main` branch.
