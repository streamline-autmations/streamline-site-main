# Streamline Automations — Marketing Site UI Kit

Pixel-close recreation of the agency homepage at **streamline-automations.agency**. Built from the production React codebase (`streamline-autmations/streamline-site-main`).

## Components

- **Navbar** — fixed top, 64px, black with 1px white/6 border
- **Hero** — Bebas Neue morphing-word headline, dual CTA, trust row
- **Marquee** — seamless client-name loop with purple diamond separator
- **Services Bento** — 3-card grid, one featured wide card
- **Why Us** — 3 pillars with role-coloured icon boxes + top-edge accents
- **How It Works** — 3 numbered steps with dashed connector
- **Tech Stack** — greyscale-to-colour logo row
- **Final CTA** — centered, orange button
- **Footer** — compact, purple accent borders

## Running

Open `index.html` directly. No build step — all JSX is transpiled inline via Babel.
